# PocketForge Marketing Copy

## Short Description (App Store Subtitle / Play Store Short Description)

**PocketForge: Code, Build, Deploy. Full-Stack Mobile IDE with Docker & AI.**

## Long Description (App Store / Play Store Full Description)

**PocketForge: The Autonomous Oracle-Architect in Your Pocket.**

Unleash the power of a full-stack development environment directly on your mobile device. PocketForge is engineered by an elite team to be the most powerful, accessible, and user-friendly mobile coding platform, outperforming all competitors by 2026. Write, test, and deploy production-ready code from your iPad, Android tablet, or even your phone.

### **Core Features Engineered for the Elite Developer:**

*   **True Cross-Platform Experience:** Built with Kotlin Multiplatform and Compose Multiplatform, PocketForge delivers a native, responsive experience across iOS, iPadOS, Android, and macOS (Catalyst).
*   **Real PTY Terminal with Docker-in-Docker:** Execute `docker run`, `docker compose up`, `kubectl`, and `ollama run` directly from a real PTY terminal. Your mobile device is now a full-fledged development server.
*   **Flexible Backend Runtimes:** Choose your power source:
    *   **PocketForge Cloud (Fly.io):** Instant Firecracker microVMs for blazing-fast, isolated environments.
    *   **Bring-Your-Own-Cloud:** Integrate seamlessly with AWS, GCP, Hetzner, and RunPod using your existing API keys.
*   **Full Offline Mode:** No internet? No problem. Ship with pre-built Alpine, Node, Python, and Go images, enabling full local container runtime on Android (Termux) and iOS/macOS (Docker Desktop).
*   **Integrated AI Agent (Claude/Grok):** A built-in Multi-Tool-Calling agent that autonomously reads/writes files, runs shell commands, performs Git operations, and debugs your code. **Your co-pilot, fully integrated.**
*   **One-Tap Deploy:** Deploy your projects instantly to Vercel, Netlify, Fly.io, Railway, and Render with automatic framework detection.
*   **Deep Git Integration:** Manage your repositories with in-app support for fork, pull request creation, commit, diff, and blame.

**PocketForge is not just an editor; it's your entire development workflow, optimized for the speed and precision of a senior engineer.**
